üìñ File I/O Basics

Python makes file operations simple with built-in functions.

Basic File Operations

# Open a file
file = open('filename.txt', 'mode')

# Perform operations
# ...

# Always close the file!
file.close()

File Modes

  • 'r' - Read (default)
  • 'w' - Write (overwrites)
  • 'a' - Append
  • 'x' - Exclusive creation
  • 'b' - Binary mode
  • 't' - Text mode (default)
  • '+' - Read and write

üìñ Reading Files

Different ways to read content from files.

Reading Entire File

# Method 1: Using read()
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Method 2: Using readlines()
with open('example.txt', 'r') as file:
    lines = file.readlines()
    for line in lines:
        print(line.strip())

Reading Line by Line

# Method 1: Using for loop (memory efficient)
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

# Method 2: Using readline()
with open('example.txt', 'r') as file:
    line = file.readline()
    while line:
        print(line.strip())
        line = file.readline()

‚úçÔ∏è Writing Files

Different ways to write content to files.

Writing to Files

# Method 1: write() - overwrites file
with open('output.txt', 'w') as file:
    file.write("Hello, World!\n")
    file.write("This is a new line.\n")

# Method 2: writelines() - writes list of lines
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open('output.txt', 'w') as file:
    file.writelines(lines)

# Method 3: Append mode
with open('output.txt', 'a') as file:
    file.write("This line is appended!\n")

Working with CSV Files

import csv

# Writing to CSV
with open('data.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Name', 'Age', 'City'])
    writer.writerow(['Alice', '25', 'New York'])
    writer.writerow(['Bob', '30', 'London'])

# Reading from CSV
with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)
